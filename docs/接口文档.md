#### 零、参数说明

1.前后端交互以唯一token作为安全标识

2.返回数据中一定包含code表示请求是否成功，下文描述返回值时省略此项

3.规定一页10条信息

#### 一、用户管理模块接口

| 接口名称     | url            | header(输入参数) | body(输入参数)                                               | param(输入参数)    | JSON(返回值)                                                 |
| ------------ | -------------- | ---------------- | ------------------------------------------------------------ | ------------------ | ------------------------------------------------------------ |
| 用户登录     | /user/login    | **string** token | **string** user_name(用户名)<br/>**string** mail(邮箱)<br/>**string** password(密码)<br/>**注意:用户名和邮箱不能同时为空、密码不能为空** |                    | 成功:<br/>**int** id(用户id)<br/>失败:<br/>**string** str(提示信息) |
| 用户注册     | /user/register | **string** token | **string** user_name(用户名)<br/>**string** mail(邮箱)<br/>**string** password(密码)<br/>**注意:用户名和邮箱、密码均不能为空** |                    | **string** str(提示信息)                                     |
| 用户注销     | /user/delete   | **string** token |                                                              | **int** id(用户id) | **string** str(提示信息)                                     |
| 获取用户信息 | /user/get/{id} | **string** token |                                                              |                    | 成功:<br/>**string** user_name(用户名)<br/>**string** mail(邮箱)<br/>**string** nick_name(昵称)<br/>**string** birthday(生日)<br/>**string** gender(性别)<br/>**string** bio(个人简介)<br/>**string** avatar_url(头像)<br/>**string** phone(电话)<br/>失败:<br/>**string** str(提示信息) |

#### 二、博客管理模块接口

| 接口名称               | url                  | header(输入参数) | body(输入参数)                                               | param(输入参数)                           | JSON(返回值)[成功/失败]                                      |
| ---------------------- | -------------------- | ---------------- | ------------------------------------------------------------ | ----------------------------------------- | ------------------------------------------------------------ |
| 写博客                 | /blog/newBlog        | **string** token | **int** author(作者id)<br/>**string** title(标题)<br/>**string** content(内容)<br/>**string** summary(简介)<br/> |                                           | **string** str(提示信息)                                     |
| 获取一篇博客           | /blog/get/{id}       | **string** token |                                                              | **int** id(博客id)                        | 成功:<br/>**int** author(作者id)<br/>**string** avatar_url(作者头像)<br/>**string** name(作者昵称或用户名)<br/>**string** bio(作者简介)<br/>**string** title(标题)<br/>**string** content(内容)<br/>**string** publish_date(发布时间)<br/>**string** update_date(更新时间)<br/>失败:<br/>**string** str(提示信息) |
| 获取一个用户的博客     | /blog/get/personal   | **string** token |                                                              | **int** id(用户id)<br/>**int** page(页数) | 成功:<br/>List {<br/>**int** id(博客id)<br/>**string** avatar_url(作者头像)<br/>**string** name(作者昵称或用户名)<br/>**string** title(标题)<br/>**string** summary(博客简介)<br/>**string** publish_date(发布时间)<br/>**string** update_date(更新时间)<br/>}<br>失败:<br/>**string** str(提示信息) |
| 获取所有用户的博客     | /blog/get/all        | **string** token |                                                              | **int** page(页数)                        | 成功:<br/>List {<br/>**int** id(博客id)<br/>**int** author(作者id)<br/>**string** avatar_url(作者头像)<br/>**string** name(作者昵称或用户名)<br/>**string** title(标题)<br/>**string** summary(博客简介)<br/>**string** publish_date(发布时间)<br/>**string** update_date(更新时间)<br/>}<br/>失败:<br/>**string** str(提示信息) |
| 获取一个用户的博客数量 | /blog/count/personal | **string** token |                                                              | **int** id(用户id)                        | 成功:<br/>**int** cnt(博客总数)<br/>**int** page(总页数)<br/>失败:<br/>**string** str(提示信息) |
| 获取所有用户的博客数量 | /blog/count/all      | **string** token |                                                              |                                           | 成功:<br/>**int** cnt(博客总数)<br/>**int** page(总页数)<br/>失败:<br/>**string** str(提示信息) |
| 删除一篇博客           | /blog/delete         | **string** token |                                                              | **int** id(博客id)                        | **string** str(提示信息)                                     |
| 修改一篇博客           | /blog/modify/{id}    | **string** token | **string** title(标题)<br/>**string** content(内容)<br/>**string** summary(简介)<br/> | **int** id(博客id)                        | **string** str(提示信息)                                     |
