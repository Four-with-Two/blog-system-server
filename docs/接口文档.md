#### 零、参数说明

1.前后端交互以唯一token作为安全标识

2.返回数据中一定包含code表示请求是否成功

3.规定一页10条信息

#### 一、用户管理模块接口

| 接口名称 | 功能                 |
| -------- | -------------------- |
| 用户登录 | 验证并返回用户标识   |
| 用户注册 | 将用户信息写入数据库 |
| 用户注销 | 删除数据库中用户信息 |

#### 二、博客管理模块接口

| 接口名称               | 功能                                 |
| ---------------------- | ------------------------------------ |
| 写博客                 | 将博客信息写入数据库                 |
| 获取一篇博客           | 从数据库中获取博客信息               |
| 获取一个用户的博客     | 从数据库中获取某个用户的一组博客信息 |
| 获取所有用户的博客     | 从数据库中获取所有用户的一组博客信息 |
| 获取一个用户的博客数量 | 计算数据库中某个用户的博客数量       |
| 获取所有用户的博客数量 | 计算数据库中所有用户的博客数量       |
| 删除一篇博客           | 删除数据库中博客信息                 |
| 修改一篇博客           | 修改数据库中博客信息                 |
